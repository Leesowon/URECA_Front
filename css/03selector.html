element selector : html tag를 선택하는 선택자 (단점 : 같은 태그 모든 곳에 적용된다.)

id selector : 특정 id를 가진 부분에만 적용

id는 html element를 고유하게 지칭하는 것 & 하나의 html 문서에 id 중복 x

class selector : 같은 css를 적용하고 싶은 부분들을 하나로 반 편성

바로 위 라인과 같은 css 적용됨

group selector : 여러 개를 묶어서 하나의 css를 적용하는 개념
